# Настройки Cursor

В этой директории содержатся файлы конфигурации и правила поведения для AI-ассистента Cursor. Эти файлы помогают настроить стиль ответов, языковые предпочтения и стандарты написания кода, а также управлять внешними инструментами (MCP).

## Структура

### `smart.mdc`
Определяет **стиль взаимодействия** и "личность" ассистента.
*   **Без "воды":** Ассистент должен давать конкретный код или объяснения сразу, избегая высокоуровневых рассуждений.
*   **Экспертный уровень:** Ответы должны быть краткими, точными и рассчитанными на опытного пользователя.
*   **Стиль:** Неформальный (casual), если не запрошено иное.
*   **Проактивность:** Предлагать решения, о которых пользователь мог не подумать.
*   **Инструкции для MCP:** Содержит закомментированные инструкции по использованию настроенных MCP инструментов (`playwright`, `sequential-thinking`). Активируются раскомментированием соответствующего блока.

### `user_rules.md`
Определяет **технические стандарты и языковые правила**.
*   **Язык общения:** Все ответы должны быть на **русском языке**.
*   **Комментарии в коде:** Исключительно на **английском языке**.
*   **Качество кода:** Приоритет чистому, эффективному и поддерживаемому коду.
*   **Принципы:** Строгое следование SOLID и KISS.

### `mcp.json`
Файл конфигурации для Model Context Protocol (MCP) серверов. Включенные серверы расширяют возможности AI-ассистента Cursor.

*   **`playwright`**: Позволяет AI взаимодействовать с веб-страницами (навигация, клики, заполнение форм, тестирование UI).
*   **`sequential-thinking`**: Направляет AI на пошаговое логическое мышление и декомпозицию сложных задач.

## Использование
Cursor автоматически использует эти файлы контекста для настройки генерации ответов, обеспечивая консистентность в разработке и коммуникации. MCP инструменты расширяют эти возможности, позволяя AI выполнять более сложные действия.

## Установка

1.  **`smart.mdc`**: Этот файл необходимо разместить по пути `.cursor/rules/smart.mdc` в корне вашего проекта. Cursor автоматически подхватит правила из этой директории.
2.  **`user_rules.md`**: Данный файл добавляется **вручную** в контекст чата (например, через `@Files` -> `user_rules.md`) или в общие настройки "Rules for AI" в интерфейсе Cursor, если вы хотите, чтобы эти правила применялись глобально.
3.  **`mcp.json`**: Файл размещается в корне проекта. MCP серверы, указанные в этом файле, запускаются автоматически при обращении к ним AI.
